A quick note on formatting (Markdown)

This text uses Markdown — a plain-text way to add headings and emphasis. The basics are all you need:

  • ## Heading — section heading. ### Subheading for one level smaller.
  • **bold** for bold, *italic* for italic.
  • Blank line between paragraphs. Lines starting with - become bullet points.

Don't worry about getting the formatting perfect — we'll clean it up. Focus on the words.
