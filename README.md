# CAF Innovation & Venture Strategy — Interactive Dataroom

An interactive website and dataroom for CAF (caf.com), the Latin American development bank, presenting our recommendations on building innovation and venture strategies for small Latin American countries.

## Status
Project scaffolding — design system defined, data integrations planned.

## Quick Start
```powershell
# (once scaffolded)
conda activate caf
python manage.py runserver
```
